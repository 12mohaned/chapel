Usage: chpl [flags|source files]

Compilation Traces:       default   type     description
-------------------       --------  -------  --------------------------------
      --print-passes      false     ->true   Print compiler passes
      --print-commands    false     ->true   Print system commands

Code Size Statistics:
---------------------
      --print-code-size   false     ->true   Print code size statistics
      --count-tokens      false     ->true   Count tokens

Optimization Control:
---------------------
      --no-bounds-checks  false     ->true   Disable bounds checking
      --no-inline         false     ->true   Do not inline functions
      --unoptimized       false     ->true   Generate naive code (many temps)

Code Generation:
----------------
      --cg-cpp-lines      false     ->true   Generate #line annotations
      --cg-chpl-lineno    false     ->true   Generate line numbers for errors
      --savec             (null)    path     Save generated C code
  -p, --parallel          true      toggle   Toggle threaded code generation

Linker Control:
---------------
  -o, --output            a.out     path     Name output executable
  -L, --lib-search-path   (null)    path     C library search path
  -l, --lib-linkage       (null)    path     C library linkage
      --ccflags           -g        string   Back-end C compiler flags

Miscellaneous Flags:
--------------------
      --instantiate-max   256       int      Limit number of instantiations
      --chplhome          .         path     Over-ride $CHPLHOME
      --devel             false     ->true   Compile as developer

Compiler Information:
---------------------
      --version                              Show Version
      --copyright                            Show Copyright
      --license                              Show License
  -h, --help                                 Help (show this list)
